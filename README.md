The `eark` package is pip-installable and is tested against python 3.6 and 3.7.

```bash
pip install eark
```  


## Using `eark`

### Running the test suite
The simplest usage of `eark` is to run the test suite. This can ensure the installation was successful.
```python
>>> import eark
>>> eark.test()
```
